## uptimectl incident resolve

Resolve an incident

```
uptimectl incident resolve [flags]
```

### Options

```
  -h, --help                 help for resolve
      --resolved-by string   User e-mail or a custom identifier of the entity that resolved the incident (default "uptimectl")
```
