## uptimectl monitors pause

Pause monitors by ID

```
uptimectl monitors pause [flags]
```

### Examples

```
# Pause a monitor
uptimectl monitors pause 1234567

# Pause multiple monitors
uptimectl monitors pause 1234567 2345678 3456789
```

### Options

```
  -h, --help   help for pause
```
